CV Generator Web Application
Project Overview
This project is a Responsive CV Generator Web Application developed using HTML, CSS, JavaScript, and Bootstrap. The purpose of this project is to help users quickly create a professional CV by entering their personal and professional information in a simple form.
The application automatically generates a formatted CV template and allows users to print or save their CV as a PDF.

Features
User-friendly form for entering information
Automatic CV generation
Responsive design using Bootstrap
Print CV or save as PDF
Simple and clean CV layout

Technologies Used
HTML5
CSS3
JavaScript
Bootstrap
